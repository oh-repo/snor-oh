# snor-oh v0.9.0 — Welcome-Back Digest + One-Click Pet Install

*Released 2026-04-22*

Two headline rolls: the mascot now greets you with a per-project summary of what changed while you were away (A9 away-digest), and the marketplace learned how to install pets straight into the app via a `snoroh://` deep link. Plus a bucket UX polish pass and a Smart Import fix that finally kills halos on transparent sprite sheets.

---

## Highlights

### 👋 Welcome-back digest

Step away from the keyboard, come back, and the mascot pops a speech bubble:

> *Welcome back! `snor-oh` +3 files, `docs-site` +1 file while you were away.*

- **Idle detection** — `UserIdleTracker` uses `SystemIdleProvider` (`CGEventSourceSecondsSinceLastEventType`) with hysteresis so a stray mouse nudge doesn't count as "back."
- **Per-project accumulation** — `AwayDigestCollector` listens on `.projectFileDelta` (emitted by `GitStatus` when the porcelain count changes) and builds a per-project rollup from `userAwayStarted` → `userReturned`.
- **Tunable** — **Settings → General → Welcome-back digest**: toggle + threshold slider (default 2 min; range 30 s – 30 min). Changes apply live.
- **Hover for detail** — each project row in the panel gains a digest tooltip while a pending digest exists. Right-click → *Clear digest* to dismiss.
- **Safety rails** — sleep-clamp prevents a ghost digest from firing right as the Mac wakes from sleep, and the `userReturned` race condition (delta arriving after return) is handled end-to-end.

### 🔗 One-click pet install from the marketplace

The marketplace can now hand a pet directly to the app. No download-then-drag-onto-app dance.

- **Deep link scheme** — `snoroh://install?id=<pet-id>` registered via `CFBundleURLTypes` (scheme `snoroh`).
- **InstallCoordinator** — on URL receipt: fetches the pet from `MarketplaceClient`, validates the `.snoroh` payload, installs into `~/.snor-oh/custom-ohhs.json` + `custom-sprites/`, and surfaces `InstallPromptView` so the user can confirm (no silent install).
- **Marketplace side** — `/install/[id]` redirects into the deep link; a fallback page at `/install-fallback` serves the plain `.snoroh` download when the app isn't installed yet, with copy-paste install instructions.
- **Gallery** — pet cards get an **Install** button alongside the existing download link.

### 🪣 Bucket polish

Smaller quality-of-life wins that add up:

- **Drag-to-reorder** cards within a bucket — position persists across relaunches.
- **Auto-collapse on focus loss** (optional) — bucket shrinks to a thumbnail when another app takes focus; snaps back on hover.
- **Per-card thumbnail cache** — no more regenerating image previews on every scroll.
- **Draggable header** — grab anywhere on the toolbar to reposition; position persists.
- **Translate-sheet dismiss fix** — the language picker sheet now dismisses cleanly after a translation completes; no more stuck-sheet state.
- **Manual auto-collapse toggle** — explicit override in Settings for users who want the bucket to stay pinned regardless of focus.

### ✂️ Smart Import — halo & edge cleanup

Imported sprites with transparent backgrounds used to ship with a faint halo of partial-alpha pixels and jagged edges. `SmartImport.swift` now:

- Detects background from corner samples with tolerance.
- Runs a dilate pass on the alpha mask to eat halos.
- Cleans jagged edges via anti-alias threshold on the alpha channel.
- Regression-covered by `Tests/SmartImportBgRemovalTests.swift` (121 lines of fixtures including the classic sprite-sheet edge cases).

## Under the hood

- `UserIdleTracker` + `IdleSecondsProvider` protocol with a `SystemIdleProvider` default — fully mockable for TDD.
- `AwayDigestCollector` is a pure value-based rollup driven by notifications — the AppDelegate just wires tracker → collector → bubble.
- `Watchdog.tick()` polls `UserIdleTracker` alongside its existing heartbeat/idle duties (no new timer).
- `MarketplaceClient` is a thin `URLSession` wrapper with 95 lines of coverage in `MarketplaceClientTests.swift`; `InstallCoordinator` has 47 lines of tests.
- `.taskCompleted` userInfo now includes the source PID — additive, no breaking changes.
- v1 `.sessionEnded` notification dropped (unused; removed in the a9 review pass).

## Updated in-app feature list

The first-launch wizard and **About** tab now lead with the bucket — reflecting its role as snor-oh's headline feature — with mascot/sessions/quick-paste following.

## Upgrade

Download the DMG from [Releases](https://github.com/thanh-dong/snor-oh/releases), drop into Applications, relaunch. Settings, custom pets, and bucket contents persist across upgrades. The ad-hoc signed build needs `xattr -cr /Applications/snor-oh.app` the first time.
